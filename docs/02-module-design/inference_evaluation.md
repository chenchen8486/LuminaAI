# 推理与评估模块设计 (Inference & Evaluation)

## 1. 功能概述
本模块负责模型训练后的验证与实际应用。提供友好的批量推理 UI，支持结果可视化叠加，并能自动生成专业的评估报告（含过漏检统计）。

## 2. ✅ 优先级
**中 (Medium)** - 紧随训练模块之后开发。

## 3. ⚠️ 潜在风险
- **推理速度**：大批量图片推理时，若不使用 Batch 推理或多线程预处理，速度可能过慢影响体验。
- **可视化性能**：在原图上叠加 Mask 或 BBox 时，若图片分辨率过高（如 4K+），OpenCV 绘图可能导致 UI 渲染延迟。

## 4. 🔍 确认点 (Checkpoints)
- [ ] **批量推理 UI**：
    - 支持导入文件夹进行批量测试。
    - 提供“结果预览墙”：网格状展示推理结果，点击可查看大图及详细置信度。
- [ ] **可视化叠加**：
    - 检测框 (BBox) 需显示类别名和置信度。
    - 分割掩码 (Mask) 需支持半透明叠加 (Alpha Blending)。
- [ ] **过漏检统计 (Evaluation)**：
    - [x] **必须支持**：自动加载对应标注文件（如有），计算 IoU。
    - [x] **必须支持**：统计并展示 False Positives (误报/过检) 和 False Negatives (漏报/漏检) 的数量及具体图片列表，方便用户针对性优化数据。
- [ ] **导出报告**：
    - 是否生成 CSV/Excel 格式的详细测试报告？
