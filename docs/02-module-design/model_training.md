# 模型训练引擎设计 (Model Training Engine)

## 1. 功能概述
本模块是系统的核心计算单元，负责封装 PyTorch 训练循环。通过多进程架构（Multiprocessing）将计算任务与 UI 主线程隔离，实时回传 Loss/Accuracy/mAP 等指标，并支持断点续训。

## 2. ✅ 优先级
**高 (High)** - 核心业务价值所在。

## 3. ⚠️ 潜在风险
- **进程通信死锁**：在 Windows 下，频繁通过 Queue 传递大量数据（如每一批次的图像 Tensor）可能导致死锁或性能下降。
- **显存管理**：用户显存不足时需优雅降级（自动减小 BatchSize 或切换 CPU），而非直接 Crash。
