# 向导模式流程设计 (Wizard Flow)

## 1. 设计理念
面向 AI 小白用户，通过“分步引导”降低操作门槛。将复杂的深度学习流程拆解为 5 个标准步骤，每一步只做一件事，确保用户不迷路。

## 2. 详细步骤规划

### Step 1: 任务选择 (Task Selection)
- **界面元素**：三个大卡片（图标+文字）
    - 🎯 **目标检测 (Object Detection)**：识别物体位置与类别（如：安全帽检测）。
    - 🎨 **图像分割 (Image Segmentation)**：像素级抠图（如：缺陷分割）。
    - 🖼️ **图像分类 (Image Classification)**：整图识别（如：良品/次品判定）。
- **动作**：用户点击卡片后，进入下一步。

### Step 2: 数据导入 (Data Import)
- **界面元素**：
    - 拖拽区域 (Drag & Drop) 或“选择文件夹”按钮。
    - 数据校验面板：自动扫描文件夹，显示“发现 100 张图片，20 个标注文件”。
    - **高级选项**：提供“格式转换”入口（如 LabelMe -> JSON）。
- **动作**：校验通过（图片与标注数量匹配或仅有图片）后，允许下一步。

### Step 3: 模型与参数 (Model & Params)
- **界面元素**：
    - **基础模型选择**：下拉菜单 (YOLOv8-Nano / ResNet-50 等)，附带“速度 vs 精度”的简单说明。
    - **显卡设置**：自动检测并推荐 (如 "NVIDIA RTX 3060 (推荐)")，支持切换 CPU。
    - **简易参数**：
        - 训练轮数 (Epochs): 滑块 (快速 50 / 均衡 100 / 深度 300)。
        - 批次大小 (Batch Size): 自动推荐 (基于显存)。
    - **高级设置** (折叠状态): 展开后可修改具体 LR, Optimizer, Weight Decay 等。
- **动作**：配置完成后，点击“准备就绪”。

### Step 4: 训练与可视化 (Training)
- **界面元素**：
    - **状态看板**：当前 Epoch / 总 Epoch，剩余时间 (ETA)。
    - **实时曲线**：Loss 下降曲线，Accuracy/mAP 上升曲线。
    - **日志窗口**：仅显示关键节点信息（如“第 10 轮评估完成，mAP=0.85”）。
- **动作**：支持“暂停”、“停止”和“提前结束”。训练完成后自动跳转。

### Step 5: 导出与验证 (Export & Verify)
- **界面元素**：
    - **模型产物**：显示最佳模型路径 (`best.pt`)。
    - **一键导出**：选择格式 (ONNX / TensorRT)，选择精度 (FP32 / FP16)。
    - **快速验证**：提供一个“上传图片测试”的按钮，直接在当前页查看推理效果。
- **动作**：点击“完成”，返回主页或打开项目文件夹。
