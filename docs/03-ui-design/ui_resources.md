# UI 资源与主题设计规范

## 1. 概述
为了提升 VisionForge (LuminaAI) 的视觉体验，赋予软件更强的“科技感”，我们引入了动态背景机制。界面背景将跟随向导步骤（Wizard Steps）自动切换，营造沉浸式的操作环境。

## 2. 资源管理
所有背景图片资源统一存放在 `ui/picture/` 目录下。

### 2.1 图片清单
目前系统内置以下背景资源（支持 `.jpg`, `.png`）：
- **1.jpg**: 首页/任务选择 (Step 1) - 营造宏大的入口感。
- **2.jpg**: 数据与配置 (Step 2 & 3) - 强调数据流与结构化。
- **3.jpg**: 模型训练 (Step 4) - 动态、算力、核心处理的视觉隐喻。
- **4.jpg**: 导出与推理 (Step 5) - 成果展示、交付感的视觉风格。

### 2.2 扩展规范
- **尺寸建议**: 1920x1080 (1080p) 或更高，以适应主流显示器。
- **格式**: JPG (推荐，体积小) 或 PNG (无损)。
- **命名**: 建议使用语义化命名（如 `bg_welcome.jpg`, `bg_training.jpg`），目前暂用数字索引。

## 3. 技术实现方案
### 3.1 背景渲染
- 采用 `QPalette` + `QBrush` (Scaled) 或 `paintEvent` 重绘机制，确保图片在窗口缩放时自适应填充（AspectFill 或 Stretch）。
- 主窗口 (`MainWindow`) 负责持有背景渲染逻辑。

### 3.2 透明度处理
- 为了让背景可见，前景的容器组件 (`WizardContainer`, `QStackedWidget` 及其子页面) 需根据需要设置半透明背景或透明背景。
- 推荐使用 `rgba(255, 255, 255, 0.85)` 作为内容区域的背景色，既保证文字可读性，又能透出底部科技感背景。

### 3.3 信号联动
- `WizardController` 发出 `page_changed` 信号。
- `MainWindow` 或 `WizardContainer` 捕获信号并切换对应的背景图片。

## 4. UI 风格指南 (Style Guide)
- **字体颜色**: 深色背景下优先使用白色/浅灰色字体；若使用半透明白底容器，则保持深色字体。
- **圆角**: 容器组件统一使用 `border-radius: 8px` 或 `10px`。
- **阴影**: 添加轻微阴影 `box-shadow` 提升层次感。
